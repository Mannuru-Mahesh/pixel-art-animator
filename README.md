# PixelForge 🎨

A beautiful, feature-rich **pixel art animator** that runs entirely in the browser — no installs needed.

🔗 **Live Demo:** [Open PixelForge](https://mannuru-mahesh.github.io/pixel-art-animator)

---

## ✨ Features

- 🖊️ **Tools** — Pencil, Eraser, Flood Fill, Eyedropper, Line, Rectangle
- 🔁 **Mirror Drawing** — Horizontal & vertical symmetry
- 🎨 **Color Palettes** — PICO-8, Game Boy, Sweetie 16, ENDESGA 32
- 🎞️ **Frame-by-Frame Animation** — Add, duplicate, delete, reorder frames
- ▶️ **Playback** — Play/pause with adjustable FPS (1–24)
- 👻 **Onion Skinning** — See previous frame ghosted behind current
- ↩️ **Undo / Redo** — 40-step history
- 📐 **Zoom** — 14 zoom levels
- 💾 **Export** — GIF animation or PNG (single frame)

## ⌨️ Keyboard Shortcuts

| Key | Action |
|---|---|
| `B` | Pencil |
| `E` | Eraser |
| `F` | Fill Bucket |
| `I` | Eyedropper |
| `L` | Line |
| `R` | Rectangle |
| `X` | Swap colors |
| `G` | Toggle Grid |
| `O` | Toggle Onion Skin |
| `Space` | Play / Pause |
| `+` / `-` | Zoom In / Out |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |

## 🚀 Running Locally

Just open `index.html` in any modern browser — no build step required!
